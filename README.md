# Full Stack Open HY – Exercises & Projects

This repository contains my solutions and projects completed as part of the **Full Stack Open** course by the University of Helsinki.

## 📚 About the Course

Full Stack Open is a free online course focused on modern web development with JavaScript. It covers building single-page applications (SPAs) using **React** and creating backend services with **Node.js** and REST APIs. The course also introduces technologies such as **Redux, MongoDB, GraphQL, and TypeScript**. ([fullstackopen.com][1])

## 🚀 What I’m Learning

* React and component-based UI development
* REST APIs with Node.js and Express
* State management with Redux
* Database integration with MongoDB
* GraphQL and TypeScript
* Testing, debugging, and application architecture

## 📁 Repository Structure

Each part of the course is organized into its own directory:

```
part_0/
part_1/
part_2/
...
```

Each folder contains the exercises and projects for that section.

## 🎯 Goal

The goal of this repository is to track my progress, reinforce learning, and build a solid foundation in full-stack web development.
